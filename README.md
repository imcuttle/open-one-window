> Open the only one window

## Installation

```bash
npm install open-one-window
# or use yarn
yarn add open-one-window
```

## Usage

```javascript
const openOneWindow = require('open-one-window')
const oneWin = openOneWindow()

oneWin.open('http://example.com')
```
